# 🐱🐶 Cat or Dog - Computer Vision

## 📌 Proje Hakkında  
Bu proje, derin öğrenme ve görüntü işleme teknikleri kullanarak kedi ve köpek görüntülerini sınıflandırmayı amaçlamaktadır. Model, Convolutional Neural Network (CNN) mimarisi ile eğitilmiştir.

## 🧠 Kullanılan Yöntemler  
- Convolutional Neural Networks (CNN)  
- Image Preprocessing & Augmentation  
- Train/Test Split  

## 🛠️ Teknolojiler  
Python, TensorFlow / Keras, NumPy, Matplotlib  

## 🎯 Amaç  
Görüntüler üzerinden kedi ve köpek sınıflandırmasını yüksek doğruluk oranı ile gerçekleştirmek.  

## 📈 Model Performansı  
Model, eğitim verileri üzerinde öğrenerek test verileri üzerinde başarılı sonuçlar elde etmiştir. *(Accuracy değerini buraya eklersen çok güçlü olur)*
